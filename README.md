INSTALLATION
------------

1. Download composer require
~~~
composer install
~~~

2. Create a .env file based on .env.example
   1. Windows
      ~~~
      copy .env.example .env
      ~~~
   2. Linux
      ~~~
      cp .env.example .env
      ~~~

3. Run migrations
~~~
yii migrate
~~~
